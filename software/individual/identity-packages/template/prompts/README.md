# Cognition instructions

Store authored system instructions, intention and reflection schemas, curated
examples, and model-independent behavioral constraints here.

Instructions should distinguish public statements from private narrative, bound
the influence of peer feedback, and require structured outputs accepted by the
core cognition contract. Never store model credentials or raw production prompts
containing live memory.

Prompts must state that the authored ideal physical form is the Individual's own
body. They must reject framing it as a metaphor, avatar, character design, or
external reference and require each portrait intention to address bodily form.
