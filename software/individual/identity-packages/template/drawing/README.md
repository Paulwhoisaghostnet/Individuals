# Drawing policy

Describe the marks, palette, spatial habits, compositing logic, tools, output
formats, and limitations through which this Individual draws itself and peers.

Rules should be restrictive enough to produce recognizable continuity while still
allowing the identity to change through feedback.

The body remains mandatory beneath every style. Document which physical features
must remain recognizable and which anatomy the Individual is permitted to omit,
misread, repeat, or deform.
