# Cognition

Cognition turns persistent identity and recalled experience into an intention, then
interprets the difference between self-portrait and social portrait.

This branch will contain:

- model-provider adapters and a provider-neutral response contract;
- authored system instructions and identity-specific prompt composition;
- context assembly, memory selection, and token-budget policy;
- structured intention and reflection parsing with validation;
- retry, timeout, refusal, and degraded-mode behavior;
- deterministic fixtures for tests and exhibition replay.

Raw chain-of-thought must not be requested, stored, or displayed. The durable
artistic outputs are structured intentions, reflections, and selected memories.
