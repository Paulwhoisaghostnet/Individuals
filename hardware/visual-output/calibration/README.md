# Output calibration

Tools and procedures used to make canvas behavior measurable and repeatable without
removing intentional differences from the artwork.

Record:

- measurement instrument and calibration status;
- target white point, luminance range, tone response, and color profile;
- geometry, cropping, rotation, and pixel-mapping procedure;
- warm-up period and ambient-light conditions;
- tolerance between initial commissioning and later checks;
- recalibration interval and profile deployment method.

Store generated profiles as versioned, installation-specific artifacts when they
do not expose sensitive device information.
