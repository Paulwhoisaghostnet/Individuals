# Canvas display

The dedicated visual surface on which one Individual publishes its current
self-portrait.

Document and validate:

- physical dimensions, aspect ratio, native resolution, and pixel density;
- panel or projection technology, finish, bezel, viewing angle, and orientation;
- brightness, black level, contrast, color gamut, and temporal behavior;
- input, cable-length, signal-recovery, and power-control behavior;
- continuous-duty rating, expected service life, and replacement availability;
- mounting datum, ventilation, status-light treatment, and service access.

Evaluation must occur in representative gallery light and from intended visitor
positions.
