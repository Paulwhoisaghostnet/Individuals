# Monitoring

Private operational visibility into hardware health without turning the public
exhibition into a status dashboard.

Document and validate:

- heartbeat, temperature, storage, memory, accelerator, camera, display, network,
  and power signals;
- thresholds based on measured normal operation;
- alert routing, escalation, acknowledgement, and quiet-hours policy;
- mapping from every signal to a location, Individual, and component role;
- retention and access controls for telemetry;
- local diagnostics that remain available during internet loss.

Monitoring must not capture visitor imagery or expose private identity prompts and
memories.
