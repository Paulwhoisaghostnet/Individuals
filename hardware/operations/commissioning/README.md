# Commissioning

The repeatable process for proving that a newly assembled installation is safe,
routed correctly, calibrated, and capable of sustained operation.

The eventual commissioning plan should include:

- inventory and connection verification;
- electrical, structural, thermal, and network inspections;
- canvas and camera calibration with routing proof for every Individual;
- full identity-loop and portrait-provenance test;
- restart, network-loss, power-loss, and component-failure tests;
- sustained burn-in under representative load;
- signed acceptance record, baselines, photographs, and known exceptions.
