# Installation host

Shared on-site compute for society scheduling, portrait routing, storage,
monitoring, and the local exhibition service.

Document and validate:

- maximum supported Individuals and expected cycle concurrency;
- storage capacity, retention policy, backup, and restoration time;
- local operation during loss of public connectivity;
- network interfaces and isolation boundaries;
- redundancy or documented degraded behavior after host failure;
- rack, thermal, power, acoustic, and remote-management requirements.

The host may coordinate Individuals but must not become the only copy of their
identity state.
