# Individual node

Compute assigned to one Individual. Record whether this boundary is a physical
device, virtual machine, container host, or accelerator allocation.

Document and validate:

- CPU, memory, storage endurance, and any GPU/NPU requirements;
- camera inputs and canvas output interfaces;
- local inference, perception, compositing, and rendering benchmarks;
- persistent identity storage and recovery behavior;
- boot, watchdog, update, and remote-service behavior;
- maximum power draw, temperature, fan noise, and enclosure clearance.

The selected node must be replaceable without replacing the Individual's durable
identity.
