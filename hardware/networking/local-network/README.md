# Local network

On-site connectivity between Individual nodes, installation services, management
systems, and the public exhibition endpoint.

Document and validate:

- logical segments, device roles, addressing, DNS, and time synchronization;
- switch port count, link speed, PoE budget if used, and expansion capacity;
- wired versus wireless decisions and interference survey results;
- firewall boundaries and permitted traffic flows;
- behavior during DHCP, DNS, switch, router, or internet failure;
- labeled patching, spare ports, configuration backup, and replacement procedure.

Prefer wired links for fixed exhibition hardware unless a documented site
constraint requires wireless operation.
