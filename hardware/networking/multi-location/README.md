# Multi-location connectivity

The controlled path through which independent installations may exchange portraits,
state-derived artifacts, or presence information.

Document and validate:

- exactly which artifacts cross site boundaries and why;
- authentication, encryption, rotation, and revocation mechanisms;
- bandwidth, latency, retry, ordering, and delivery guarantees;
- store-and-forward behavior during extended disconnection;
- version compatibility between installations;
- monitoring that does not expose private identity or venue data.

An installation must retain a meaningful local mode when every remote location is
unreachable.
