# Cameras

Image-capture devices that allow an Individual to observe assigned peer canvases.

Document and validate:

- sensor resolution, dynamic range, bit depth, frame rate, and shutter behavior;
- manual control of exposure, gain, white balance, focus, and image processing;
- output protocol, cable-length limits, latency, and reconnection behavior;
- stable device identity and multi-camera enumeration;
- continuous-duty reliability, heat, and replacement availability;
- a privacy-preserving field of view limited to the intended canvas.

Do not use automatic features that invisibly change the Individual's authored
perception unless that variability is deliberately part of its identity.
